The Actor Framework files are saved for LabVIEW 2011.
To install the Actor Framework, follow these steps:

1. Extract the contents of ActorFramework.zip into the following directory:
	<labview>\vi.lib\ActorFramework
2. Copy the three .mnu files to the following directory:
	<labview>\menus\Categories\Data Communication

The Actor Framework is a software library containing classes that supports the writing of applications in which multiple VIs run independently while communicating with each other.
For detailed information on how to use the Actor Framework, refer to the document below on ni.com:

Read This First to get started with Actor Framework
